Proyecto

Mi proyecto como aprendiz consiste en: Crear un software que ayude a las personas con discapacidad visual absoluta
a manejar un dispositivo móvil o un computador lo más parecido a como lo hace una persona que puede ver correctamente.

Como es un proyecto "ambicioso", que va a requerir de mucho tiempo y conocimiento, quizás no se lleve el desarrollo absoluto del software al finalizar el tecnólogo, pero si un software funcional.
